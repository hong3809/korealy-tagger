1
제품 등록 / 업데이트 감지

2
korealy 공급업체 필터링
Vendor가 "korealy"인 제품만 처리 (대소문자 무관)
3
필수 태그 추출 (4가지)
Product Title 키워드 · Product Type · Vendor · Collection 자동 추가
4
Description 분석
HTML Description에서 성분·효능·피부타입·특성 키워드 추출
5
기존 태그 유지 + 병합
기존 태그를 보존하고 새 태그와 중복 제거 후 병합
6
Shopify Admin API로 업데이트
최종 태그 목록을 Shopify에 즉시 반영
